# PvZRp-Repacker

VERY Early and Undercooked™️Release of PvZ: Replanter

Repack Utility for PvZ: Replanter, a tool for replacing assets inside the bundles of Plants Vs. Zombies Replanted

This tool can automate the:
  - Repacking of asset files back into a bundle

Still work in progress, of course.
It's very hacky, so make sure to back up your files before using this tool
